# 🐶Roadpet
반려동물의 기록을 작성하고 정보를 공유하는 커뮤니티

<br>

## 🔎프로젝트 소개
반려동물의 대한 다양한 서비스를 제공하는 프로젝트로, 반려동물에 대한 산책기록을 작성하고 사람들과 공유할 수 있습니다.
또한 유기동물찾기 서비스를 통해 실종위치를 등록하고 다른 유저들과 항목을 공유하며 구조를 도울수 있습니다.
기본적인 반려동물 커뮤니티 기능을 제공하며 여러 정보를 다른 유저들과 공유할 수 있습니다.
